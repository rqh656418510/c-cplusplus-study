### 项目开发工具

| 软件               | 版本        | 说明                                  |
| ------------------ | ----------- | ------------------------------------- |
| C++ 标准           | `11`        | C++ 标准的版本                        |
| G++（GCC）         | `12.2.0`    | 建议使用 `9` 版本的 G++（GCC） 编译器 |
| CMake              | `3.25.1`    | C/C++ 项目构建工具                    |
| Linux              | `Debian 12` | Muduo 库不支持 Windows 平台           |
| Visual Studio Code | `1.100.2`   | 使用 VSCode 远程开发特性              |
